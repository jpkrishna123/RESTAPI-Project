How to Build
=============

mvn clean install

How to Run
===========

java -jar target/orders-0.0.1-SNAPSHOT.jar

URL Details
==============

List Orders: [GET] http://localhost:8088/orders

Fetch Order Details: [GET] http://localhost:8088/orders/{order_id}

List Products: [GET] http://localhost:8088/products

Fetch Product Details: [GET] http://localhost:8088/products/{product_id}

Note: post man collections was checked into docs folder
